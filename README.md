  👋 Hey, I'm David                                                                                     
                                                                                                        
  I build **agentic systems** and production-grade platforms at the intersection of **AI, fintech & payments** — 
  from data pipelines and backend services to cross-platform products.                                                                                        
                                                                                                        
  Most days you'll find me:                                                                             
  - Designing scalable data pipelines (Spark, Airflow, Snowflake, Kafka, Databricks)
  - Designing **agentic / LLM systems** — multi-agent orchestration, RAG, tool-calling and evals
  - Shipping backend services in Kotlin / Spring Boot
  - Building cross-platform products with Flutter and Next.js
  - Wiring up payment flows, multi-currency pricing, and marketplace infrastructure
  - Coordinating multi-repo systems (backend → cloud functions → mobile → web)
  - Obsessing over clean abstractions, idempotency, and systems that don't page you at 3am

  Currently building: **[GuideX](https://lets.guidex.you)** — a travel-tech experiences marketplace, end-to-end — Kotlin/Spring Boot APIs (hosted on Cloud Run), Firebase Cloud
  Functions, Data Connect, Flutter mobile app, and a Next.js web client — with real-time bookings, ticketing, and
  international payment processing.

  Infra - Terraform, GCP, AWS, Azure

  ### 🚀 Selected projects
  - **[GuideX](https://lets.guidex.you)** — a travel-tech experiences marketplace, “where the world books Africa.” Live on iOS, Android & web.
  - **DXJ Signal Engine** — a weekly AI equity-research engine: a deterministic signal flags high-growth US stocks; a “Hermes” deep-research agent writes cited dossiers on the top names.
  - **Cygnal** — a real-time fraud intelligence platform for African banks & fintechs (ISO 8583, rules + ML, examiner-grade audit trail).
  - **Bosun** — a private, always-on agent that orchestrates a crew of coding agents across projects, with a tap-to-approve safety gate.

  AI-Native Development

  I write code with Cursor, Claude Code, and OpenAI Codex as core parts of my workflow — not as
  autocomplete, but as genuine force multipliers:

  - Cursor as my primary IDE — inline AI assistance for rapid iteration across Kotlin, Dart, TypeScript,
   and React
  - Claude Code as a pair programmer for complex, multi-repo tasks — I architect the plan, then execute
  full-stack features spanning 4 codebases in a single session (backend → cloud functions → mobile →
  web)
  - OpenAI Codex for autonomous task execution and parallel workstreams
  - AI-assisted cross-codebase debugging — tracing issues across service boundaries (e.g. tracking why a
   Firestore document isn't created from Spring Boot through Cloud Functions to the mobile client)
  - AI-driven merge conflict resolution on complex domain logic files
  - Comprehensive PR generation with structured descriptions across multiple repositories

  I treat AI tooling the way I treat any good infrastructure: learn its strengths, set clear boundaries,
   and let it handle the throughput while I focus on the architecture and product decisions.

  I care deeply about robustness, clarity, and shipping things that actually get used.
